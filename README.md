> Nicer way to manage your cookies

## Install

```sh
yarn install @beeeku/cookieplus
```

### Importing library

```javascript
import { getCookie, setCookie, removeCookie, clearCookie } from 'mylib';
```

### Usage

```javascript
const times = parseInt(getCookie('times'), 10) || 0;
setCookie('times', times + 1);
removeCookie('times');
clearCookie();
```
